A port is a structural element of a system that can be connected to a signal.

Connect this port to another port.

Parameters

other – The other port to connect to

Raises

  • ShapeMismatchError – if the shapes of the ports do not match

  • MultipleSignalsError – if both ports are already connected to different signals
