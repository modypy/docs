The Blocks LibraryÔÉÅ

The Blocks Library provides re-usable system elements:

Sources (modypy.blocks.sources)

are blocks and signals that provide possibly time-dependent input data. A specific variant is the (modypy.blocks.sources.constant) signal, which provides a constant value.

Linear blocks (modypy.blocks.linear)

provide building-blocks for linear systems, such as the modypy.blocks.linear.LTISystem class for linear time-invariant systems in the typical state-space formulation.

Electro-mechanical blocks (modypy.blocks.elmech)

are useful for modelling electro-mechanical systems. The main constituent of this module currently is the modypy.blocks.elmech.DCMotor class for modelling DC-motors.

Aerodynamic blocks (modypy.blocks.aerodyn)

are blocks and signals for aerodynamic modelling. Most prominent is the modypy.blocks.aerodyn.Propeller class for modelling a propeller in static configuration, and the modpyp.blocks.aerodyn.Thruster class to model a directional thruster with thrust and torque.

Rigid-body motion blocks (modypy.blocks.rigid)

allow modelling the translational and rotational motion of rigid bodies.
