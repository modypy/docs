class modypy.blocks.linear.LTISystem(parent, system_matrix, input_matrix, output_matrix, feed_through_matrix, initial_condition=None)[source]

Implementation of a linear, time-invariant block of the following format:

dx/dt = system_matrix * x + input_matrix * u y = output_matrix * x + feed_through_matrix * u

The matrices system_matrix, input_matrix, output_matrix and feed_through_matrix define the state and output behaviour of the block.

class modypy.blocks.linear.Sum(parent, channel_weights, output_size=1)[source]

A linear weighted sum block.

This block may have a number of inputs which are interpreted as vectors of common dimension. The output of the block is calculated as the weighted sum of the inputs.

The channel_weights give the factors by which the individual channels are weighted in the sum.

This class is deprecated. Use sum_signal instead.

modypy.blocks.linear.sum_signal(input_signals, gains=None)[source]

Create a signal that represents the sum of the input signals multiplied by the given gains.

The signals must have the same shape and there must be exactly as many entries in the gains tuple as there are input signals.

Parameters

  • gains – A tuple of gains for the input signals. Optional: Default value is all ones.

Returns
