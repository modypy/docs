Linearization

Provides functions to determine the jacobi matrix for linearizing the system around a given state with specified inputs.

modypy.linearization.system_jacobian(config: modypy.linearization.LinearizationConfiguration, single_matrix=False)[source]

Numerically determine the jacobian of the system at the given state and input setting.

The function uses polygonal interpolation of the given order on the components of the system derivative and output function, choosing interpolation points at a given distance from the given state and input values.

This can be used in conjunction with find_steady_state to determine an LTI approximating the behaviour around the steady state.

NOTE: This function currently does not honor clocks.

Parameters

  • config – The LinearizationConfiguration instance describing the linearization to be performed

  • single_matrix – Flag indicating whether a single matrix shall be returned. The default is False.

Returns

the jacobian, if single_matrix is True, and a tuple of system matrix, input matrix, output matrix and feed-through matrix, if single_matrix is False.

Raises

ValueError if the system does not have states or inputs –
